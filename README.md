Program for the purpose of searching Victoria, Australia for plots of land that match certain characteristics (area, frontage, boundary fall, location etc.). \
\
Not intended for use by anyone aside from myself (so proceed at your own risk). \
\
**Run instructions:** \
**1.** Request vicmap plots, vicmap planning, vicmap address datasets from https://datashare.maps.vic.gov.au/. \
&nbsp;&nbsp;&nbsp;&nbsp;**1a.** File format should be: ESRI File Geodatabase, MGA 2020 Zone 55, No buffer. \
\
**3.** Unzip and place the VMADD.gdb, VMPLAN.gdb and VMPROP.gdb folders into a "data/raw" directory within the program files. \
\
**4.** Filters can be configured on the config.py script. Currently the boundary fall filter does not work. \
